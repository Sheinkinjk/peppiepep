## Troubleshooting

### If you see: "Email service not configured"

**Problem:** Environment variables not loaded

**Solution:**
1. Go back to Vercel â†’ Settings â†’ Environment Variables
2. Verify both RESEND_API_KEY and RESEND_FROM_EMAIL are there
3. Redeploy the project (Deployments â†’ ... â†’ Redeploy)
4. Wait 2 minutes for deployment
5. Try again

---

### If you see: "Failed to send email"

**Problem:** Customer has no email address OR API key is wrong

**Solution:**
1. Check customer has valid email in Clients & Ambassadors tab
2. Verify API key was copied correctly (no extra spaces)
3. Check Resend dashboard for error logs: https://resend.com/emails

---

### If emails go to spam

**Problem:** Domain not verified (normal for first emails)

**Solution:**
1. This is normal! First few emails may go to spam
2. For better deliverability, verify your domain (see full guide)
3. Check spam/junk folder for test emails
